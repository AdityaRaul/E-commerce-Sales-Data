# E-commerce-Sales-Data
This project is an interactive Ecommerce Sales Dashboard developed using Microsoft Power BI to analyze sales performance, customer behavior, profit trends, and payment insights. The dashboard transforms raw ecommerce data into meaningful visual reports that help in better business decision-making.

The dashboard includes KPI cards, bar charts, donut charts, and interactive filters for analyzing sales across different states, categories, customers, and payment modes. It provides detailed insights into total sales amount, quantity sold, overall profit, and average order value (AOV).

Key business insights such as top-performing states, profitable product categories, monthly profit trends, and customer purchasing patterns are visualized using dynamic and user-friendly reports. The project demonstrates skills in data visualization, dashboard design, data analysis, and business intelligence using Power BI and DAX.
